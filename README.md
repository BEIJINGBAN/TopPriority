# Apex Task-Master | 极简桌面任务大师 🚀

> **项目背景**：在电脑前办公时，思维常被琐事或社交软件打断。我需要一个“永远钉在视野内”的任务清单，强制自己保持专注。于是开发了这款**始终置顶**、**具备优先级矩阵**的任务管理工具，确保核心目标永远在视线中央。

## ✨ 核心特性

* **📌 永远置顶 (Always on Top)**：真正的“桌面级”提醒，无论打开多少网页或文档，它始终可见。
* **⚖️ EIS 优先级算法**：
    * **重要度 (Importance)**：从“微小”到“关键”，决定任务的本质价值。
    * **紧急度 (Urgency)**：从“闲暇”到“🔥紧急”，决定处理的先后顺序。
* **🔒 专注锁定模式**：一键锁定界面，禁用所有操作按钮，防止高强度工作时的误触。
* **📜 自动化复盘**：任务完成后自动移入历史库，支持按日期快速检索。
* **💾 纯本地存储**：数据仅存储在本地 `apex_tasks.json`，无需联网，隐私极度安全。

## 🖱️ 交互技巧

1. **快速编辑**：**双击**任何一条任务的文字，即可快速弹出修改窗口。
2. **强制钉选**：点击任务右侧的 **📌 图标**，该任务将无视算法，永久锁定在列表最上方。
3. **窗口拖动**：点击顶部的**时间区域**或**日期区域**，即可按住并拖动整个窗口。
4. **智能删除**：点击垃圾桶图标即可移除不再需要的任务。

## 🛠️ 如何使用

### 方式 A：直接运行 (Windows)
* 下载仓库中的 `ApexTaskMaster.exe`。
* 直接双击启动，无需配置任何 Python 环境。

### 方式 B：通过源码运行
```bash
# 1. 确保安装了 Python 3.x
# 2. 直接运行脚本
python ApexTaskMaster.py
```
